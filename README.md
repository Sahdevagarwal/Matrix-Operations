- Interactive console applications

- Matrix Addition

## How to Run

### 1. Clone the repository

```bash
git clone <your-repo-link>
```

### 2. Navigate to the project folder

```bash
cd matrix-operations
```

### 3. Install NumPy

```bash
pip install numpy
```

### 4. Run the program

```bash
python main.py
```
